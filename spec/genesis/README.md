# Genesis artifacts

The origin material this project was built from, archived here for provenance.
Nothing in the build depends on these files — they are the record of where it
started, not part of the running system.

## Orchestration prompts

- `regel-session-prompt.md` — the original kickoff (2026-07-09): design →
  adversarial evaluation → build, as handed to the prime orchestrator.
- `regel-stage-e-session-prompt.md` — the Stage E handoff (2026-07-16): review,
  proof CRM, claim-evidence, final report.

(The Stage F residue burn-down that followed was driven by a third prompt given
inline in that session; it is summarized in `spec/STATE.md` and
`spec/gates/STAGE-F.md` rather than reproduced here.)
